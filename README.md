# VibeHotel

Full-stack hotel website. Frontend built with Vite + React + Tailwind CSS. Backend built with Express + TypeScript + Supabase.

> UI template generated by [Magic Patterns](https://magicpatterns.com): [Source Design](https://www.magicpatterns.com/c/sm36mmatowmd9gksme6x2c)

## Project Structure

```
VibeHotel/
├── frontend/   — Vite + React + Tailwind CSS
└── backend/    — Express + TypeScript + Supabase
```

## Getting Started

### 1. Configure environment variables

```bash
# Backend
cp backend/.env.example backend/.env
# Fill in SUPABASE_URL, SUPABASE_ANON_KEY, SUPABASE_SERVICE_ROLE_KEY

# Frontend
cp frontend/.env.example frontend/.env
# Fill in VITE_SUPABASE_URL, VITE_SUPABASE_ANON_KEY
```

### 2. Install dependencies

```bash
npm run install:all
```

### 3. Run both frontend and backend

```bash
npm run dev
```

Or separately:

```bash
npm run dev:frontend   # http://localhost:5173
npm run dev:backend    # http://localhost:4000
```
